```sh

```

## Documentation

### Endpoints
##### Basic summary

* GET - {url}/api/customers - **Returns all rows**
* GET - {url}/api/customers/[id] - **Returns a row by id**
* POST - {url}/api/customers - **Add a record**
* PUT - {url}/api/customers/[id] - **Update row by id**
